# Delivery concept

Each component delivers through a pull request that changes only its owned path.

Integration-owned zones:
- contracts/
- deploy/
- packaging/
- .github/

Canonical component paths:
- components/scale-radio-starter/
- components/scale-radio-tuner/
- components/scale-radio-autoswitch/
- components/scale-radio-bridge/
- components/scale-radio-fun-line/
